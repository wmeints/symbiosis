## When to use this skill

- When creating or editing React component files (.jsx, .tsx) in functional or class component patterns
- When building Vue components (.vue files) with composition or options API
- When developing Svelte components (.svelte files) or Angular components (.ts with templates)
- When designing reusable UI component libraries or design system components
- When implementing component composition by combining smaller components into complex UIs
- When defining component props, interfaces, or prop types for component APIs
- When managing component state with hooks, computed properties, or reactive declarations
- When working in component directories like components/, src/components/, or lib/components/
- When refactoring monolithic components into smaller, focused single-responsibility components
- When documenting component usage, props, and providing usage examples
