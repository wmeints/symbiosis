## When to use this skill

- When writing or editing CSS files (.css, .scss, .sass) for styling user interfaces
- When applying Tailwind utility classes in HTML templates, JSX, or component files
- When using CSS Modules (.module.css) for scoped component styling
- When writing styled-components, Emotion, or other CSS-in-JS solutions in JavaScript/TypeScript files
- When defining design tokens for colors, spacing, typography, and other design system values
- When implementing BEM naming conventions for CSS classes
- When configuring theme variables or customizing framework default styles
- When optimizing CSS for production with purging or tree-shaking to remove unused styles
- When working with framework-specific styling patterns and avoiding excessive style overrides
