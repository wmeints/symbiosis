## When to use this skill

- When creating or editing React, Vue, Svelte, or Angular component files that render UI elements
- When writing HTML templates, JSX, TSX, or other frontend markup languages
- When implementing forms with input fields, labels, validation messages, and error states
- When building interactive elements like buttons, links, dropdowns, modals, or dialogs
- When creating navigation menus, sidebars, or header components that users navigate with keyboard
- When adding images and need to provide alternative text for screen readers
- When implementing focus management in single-page applications or dynamic content
- When ensuring proper heading hierarchy and semantic structure in page layouts
- When configuring ARIA attributes for complex widgets or custom interactive components
- When testing color contrast ratios or ensuring information isn't conveyed by color alone
