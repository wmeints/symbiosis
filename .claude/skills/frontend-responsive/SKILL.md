## When to use this skill

- When implementing mobile-first designs that start with mobile layouts and enhance for larger screens
- When using CSS media queries to define responsive breakpoints for mobile, tablet, and desktop
- When creating fluid layouts with percentage-based widths, CSS Grid, or Flexbox containers
- When applying responsive utility classes in frameworks like Tailwind (sm:, md:, lg:, xl: prefixes)
- When using relative units (rem, em) instead of fixed pixels for better scalability
- When testing UI components across multiple screen sizes from mobile to desktop
- When ensuring tap targets meet minimum size requirements (44x44px) for mobile touch interfaces
- When optimizing images and assets for different screen sizes and network conditions
- When implementing content priority strategies to show important content first on smaller screens
- When ensuring readable typography across all viewport sizes without requiring zoom
